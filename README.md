#### Installation

1. 克隆仓库到本地
2. 将yubai-qiduoduo目录下的qiduoduo数据文件夹移动到本地MySql的data目录
3. 进入qiduoduo-serve目录，运行index.js文件，启动后端
4. 进入vue/init前端项目目录,运行命令: npm i 加载依赖， 运行命令：npm run dev 启动前台项目
5. 进入localhost:3000，进行登录操作
